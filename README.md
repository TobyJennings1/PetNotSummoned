# Pet Not Summoned

Shows a persistent icon and raid warning when your combat pet is not active.

## Supported Classes
- Hunter
- Warlock
- Unholy Death Knight

## Features
- Persistent icon when pet missing
- Raid warning text (out of combat)
- Sound alert
- Spec-aware detection for Unholy DK

## Installation
Place the `PetNotSummoned` folder inside:
World of Warcraft/_retail_/Interface/AddOns/
